# Python Project Setup

This repository uses a dedicated setup script to ensure you never bloat your global Python installation.

## Prerequisites
* **Python 3.11+** installed on your system.

## One-Command Setup

To create the virtual environment and install all dependencies from `requirements.txt`, open your terminal at the project root and run:

### macOS / Linux
```bash
python3 setup_env.py
```

### Windows
```cmd
python setup_env.py
```

## Activating the Environment

The setup script will output the exact command you need to activate the environment. Once activated, your terminal prompt will change, and you can run the project safely!
